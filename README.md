# GCSP Portfolio (static)

This is a minimal static portfolio skeleton for a Grand Challenges Scholars Program (GCSP) student.

Files created:
- `index.html` — Home / About page
- `entrepreneurship.html`, `interdisciplinary.html`, `multicultural.html`, `research.html`, `service-learning.html`, `gcsp-experience.html` — placeholder pages
- `assets/css/styles.css` — global styles
- `assets/images/profile.svg` — placeholder profile image

Open `index.html` in a browser to view the About page. The other pages are intentionally blank placeholders and will be filled with content later.
